# 🏥 PulseFlow – Healthcare Patient Health Records Dashboard

 PulseFlow is a modern healthcare analytics dashboard project designed to manage, clean, transform, explore, model, and visualize patient health records in one place. It supports role-based login, custom data upload, risk analysis, alerts, insights, and interactive dashboards for healthcare data exploration.
--------------------------------------------------------------------
🚀 Visit the Live Project: https://pulseflowhealthcare.netlify.app/
--------------------------------------------------------------------

## ✨ Features

- 🔐 **Role-Based Login System**  
  Supports separate login access for **Admin, Doctor, Nurse, and Analyst** users.

- 📤 **Custom Data Upload**  
  Upload your own patient data using **CSV** or **Excel** files.

- 🧹 **Data Cleaning**  
  Handles missing values, duplicates, invalid formats, and inconsistent records.

- 🔄 **Data Transformation**  
  Creates derived fields like age groups, risk score, length of stay, and readmission flags.

- 🔍 **Data Exploration**  
  Displays summary statistics, distributions, trends, and abnormal patterns.

- 🤖 **Data Modeling**  
  Supports simple predictive analysis such as risk prediction and alert detection.

- 💡 **Insight Generation**  
  Produces key insights, anomalies, and short natural-language summaries.

- 📊 **Interactive Dashboard**  
  Includes patient KPI cards, trend charts, breakdowns, filters, and detailed tables.

- 👤 **Patient Profile View**  
  Shows complete individual patient history, vitals, diagnosis, medication, and visits.

- 🔎 **Search & Filters**  
  Search by patient ID, name, doctor, diagnosis, department, or date.

- ⚠️ **Abnormal Alerts**  
  Highlights critical values such as high BP, low SpO2, abnormal labs, and missed follow-ups.

- 🆚 **Comparison View**  
  Compare multiple patients side by side.

- 📁 **Export Reports**  
  Download filtered reports as CSV, Excel, or PDF.

- 🎨 **Light/Dark Theme Toggle**  
  Switch between light and dark mode with saved theme preference.

- 🖥️ **Modern Healthcare-Tech UI**  
  Features a professional background combining medical and technology elements.

## 🧠 Workflow

1. **Login**
2. **Upload Data**
3. **Clean Data**
4. **Transform Data**
5. **Explore Data**
6. **Model Data**
7. **Generate Insights**
8. **View Dashboard**

The dashboard is displayed only after all data preparation steps are completed.

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript / Streamlit / React
- **Data Processing:** Python, Pandas, NumPy
- **Visualization:** Plotly / Matplotlib / Seaborn
- **Optional Backend:** Flask / FastAPI
- **File Support:** CSV, Excel
- **Authentication:** Role-based login system
